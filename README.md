# TECHSOURCEDGE - IT Consulting Services Website

A modern, responsive website for IT consulting services built with React, Vite, and Tailwind CSS.

## Features

- 🎯 Responsive navigation with mobile menu
- 🎨 Auto-rotating carousel/hero section
- 💼 Services showcase grid (5 columns x 2 rows)
- 📝 About section with company information
- 📧 Comprehensive footer with links and contact info

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Lucide React (icons)

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open your browser and visit `http://localhost:5173`

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint

## Project Structure

```
src/
├── components/        # React components
│   ├── Navbar.jsx    # Navigation bar
│   ├── Carousel.jsx  # Hero carousel
│   ├── Services.jsx  # Services grid
│   ├── About.jsx     # About section
│   └── Footer.jsx    # Footer component
├── App.jsx           # Main app component
├── main.jsx          # Entry point
└── index.css         # Global styles
```

## Build for Production

```bash
npm run build
```

The built files will be in the `dist` directory.
